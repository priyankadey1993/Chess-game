# Chess-game
A lightweight, beautiful, and fully interactive Chess Board Game built using the Qt6 framework and QML. This project focuses on a clean UI, smooth piece movements, and modern layout management without relying on heavy external modules.

Features
Modern UI: Designed with sleek, alternate dark-green and cream squares inspired by modern chess platforms.
Full Piece Set: Features all 32 standard chess pieces represented cleanly using high-quality unicode symbols (Emojis).
2D Matrix Logic: Dynamic state management using an $8 \times 8$ multidimensional JavaScript array in QML.
Interactive Controls: Implements a two-click selection and movement mechanism via QML MouseArea.
Highlight System: Active pieces are instantly highlighted to give clear visual feedback during a move.

Tech Stack
Framework: Qt 6.x
Frontend: QML (Qt Quick)
Build System: CMake
